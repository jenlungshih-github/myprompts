# 🍌 My Prompts v0.5.0

## 🚀 技術細節 (Technical Details)

本專案採用現代化的 Web 技術棧開發：

-   **Frontend Framework**: React 19
-   **Build Tool**: Vite 7
-   **Styling**: Vanilla CSS (Nanobana Pro 風格)
-   **Backend / AI**: Google Generative AI SDK (動態模型選取，直連 Gemini API)
-   **Database**: Firebase Firestore (即時同步自定義提示詞)
-   **Storage**: Firebase Cloud Storage (用於儲存提示詞參考圖片)
-   **Deployment**: Firebase Hosting
-   **State Management**: React Hooks (useState, useEffect)
-   **Stability Features**: 
    - 動態 API 模型探索 (自動選取最佳可用 Flash 模型)
    - 健壯的 JSON 解析與 Markdown 清洗
-   **Security**: Client-side API Key management (LocalStorage)
-   **Local Storage**: Browser LocalStorage (僅用於儲存 API Key)
